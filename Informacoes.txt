Quando fazemos o git clone do repositório remoto do GITHUB...

não precisamos realizar o comando git init... só faremos isso quando criamos no
repositório local... desde o começo

Como Baixamos a pasta do GITHUB, então basta fazer o git add do arquivo criado...
e realizar o git commit e o git push...

git add informações.txt

não coloque o nome do arquivo com caracteres especiais...
senão não vai conseguir adicionar no git add
